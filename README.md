# Vuer

> A brazilian helper bot for vuejs-br community and GitHub App built with [Probot](https://github.com/probot/probot)

## Setup

```sh
# Install dependencies
yarn install

# Run the bot
yarn start
```

The master branch is automatically deployed to heroku on new commits.
